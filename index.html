profile

1983年3月5日の長野県松本市生まれ。
2009年Web系専門学校卒業後、有限会社ワンダフルに入社。HTML / CSSコーディングからjavascriptの演出・Ajaxまでフロントエンド部分を担当。
2013年はjsを使った様々な演出を案件で経験、2014年はデザインとインタラクションの繋げ役になろうと画策しています。

Born in Matsumoto, Nagano Prefecture on March 5, 1983.
After graduating a web based vocational school in 2009, I joined Wonderful ltd. I am a leader of technical department of the web team. My strengh lies in HTML/CSS and interractions using javascript and Ajax. I have been studying the issues experienced in various production that uses a js in 2013, to become a connected role of interaction and web design in 2014.

1983 년 3 월 5 일 나가노 출생.
2009 년 Web 계 전문 학교 졸업 후, 유한 회사 원더풀 입사. HTML / CSS 코딩에서 javascript의 연출 · Ajax까지 프론트 엔드 부분을 담당. 2013 년은 js를 사용한 다양한 연출을 안건으로 경험 2014 년은 web 디자인과 상호 작용 연결해 역이 되려고 공부하고 있습니다.
